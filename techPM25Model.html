Web Application: การคำนวณค่า PM2.5

ขั้นตอนการทำงานโดยละเอียดของฟังก์ชัน JavaScript calculatePM25()

  1. เริ่มต้นด้วยค่าฝุ่นพื้นฐาน (Base Value)

    การคำนวณจะเริ่มต้นจากค่าฝุ่นพื้นฐานที่ 15 µg/m³ ซึ่งเปรียบเสมือนปริมาณฝุ่นในวันปกติของพื้นที่เมือง ที่ยังไม่มีปัจจัยรุนแรงอื่นๆ เข้ามาเกี่ยวข้อง

    let basePM25 = 15;
  2. นำค่าจากผู้ใช้มาคำนวณผลกระทบ (Effect Calculation)

    สคริปต์จะดึงค่าที่ผู้ใช้ปรับจากฟอร์มทั้งหมด แล้วนำมาคำนวณ "ผลกระทบ" ของแต่ละปัจจัย เพื่อนำไปบวกหรือลบกับค่าพื้นฐาน

    • ความเร็วลม (Wind Effect):
      let windEffect = -0.7 * wind;

      เป็นความสัมพันธ์แบบ "ผกผัน" (ค่าเป็นลบ) หมายความว่า ยิ่งลมแรง ค่าฝุ่นยิ่งลดลง เพราะลมช่วยพัดพาฝุ่นให้กระจายตัว

    • ปริมาณน้ำฝน (Rain Effect):
      let rainEffect = -20 * rain;

      เป็นความสัมพันธ์แบบ "ผกผัน" ที่รุนแรงมาก (ตัวคูณสูงถึง -20) เพราะฝนมีประสิทธิภาพสูงในการชะล้างฝุ่นละอองออกจากอากาศ

    • สภาวะอุณหภูมิผกผัน (Inversion Effect):
      let inversionEffect = inversion ? 35 : 0;

      เป็นการตรวจสอบเงื่อนไข ถ้าผู้ใช้ติ๊กช่อง "อากาศปิด" (inversion เป็น true) จะบวกค่าฝุ่นเพิ่มขึ้นทันที 35 µg/m³ เพราะสภาวะนี้จะกดอากาศและกักฝุ่นไว้ แต่ถ้าไม่ติ๊ก ก็จะไม่มีผลกระทบ (เป็น 0)

    • การเผาในภาคเกษตร (Burning Effect):
      let burningEffect = burning * 20;

      เป็นความสัมพันธ์แบบ "โดยตรง" (ค่าเป็นบวก) ยิ่งระดับการเผาสูง ค่าฝุ่นยิ่งเพิ่มขึ้นมาก เพราะเป็นแหล่งกำเนิดฝุ่นโดยตรง ตัวคูณที่สูง (20) สะท้อนว่าเป็นปัจจัยหลักที่ก่อให้เกิดมลพิษสูง

    • ปริมาณการจราจร (Traffic Effect):
      let trafficEffect = traffic * 8;

      เป็นความสัมพันธ์แบบ "โดยตรง" เช่นกัน ยิ่งการจราจรหนาแน่น ค่าฝุ่นยิ่งเพิ่มขึ้น ซึ่งเป็นแหล่งกำเนิดฝุ่นที่สำคัญในเขตเมือง

  3. รวมผลลัพธ์ทั้งหมด

    นำค่าพื้นฐานมาบวกรวมกับผลกระทบของทุกปัจจัย (ซึ่งบางค่าก็เป็นค่าลบ)

    let totalPM25 = basePM25 + windEffect + rainEffect + inversionEffect + burningEffect + trafficEffect;
  4. ปรับค่าสุดท้ายและแสดงผล

    เนื่องจากค่าฝุ่นไม่สามารถติดลบได้ จึงมีการตรวจสอบเพื่อให้แน่ใจว่าค่าต่ำสุดคือ 0 จากนั้นจึงปัดเศษให้เป็นจำนวนเต็ม และส่งค่าที่คำนวณได้ไปอัปเดตตัวเลข, ระดับคุณภาพอากาศ, และคำแนะนำบนหน้าเว็บ

สรุป

แบบจำลองนี้ใช้ "สมการเชิงเส้นอย่างง่าย" โดยมีค่าเริ่มต้น แล้วปรับเพิ่ม-ลดตามความรุนแรงของแต่ละปัจจัยที่เราป้อนเข้าไป เพื่อจำลองให้เห็นว่าปัจจัยใดมีอิทธิพลต่อค่าฝุ่น PM2.5 มากน้อยเพียงใด
