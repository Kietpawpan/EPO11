RACI Chart

แผนผังแสดงบทบาทและความรับผิดชอบ | โครงการจัดการขยะมูลฝอย (CCE) นครชัยบุรินทร์

กิจกรรม (Task) ผู้จัดการโครงการ
(สคพ.11)		 ทีมวิชาการ
(สคพ.11)		 ผู้ประสานงานพื้นที่
(อปท./ทสจ.)		 ภาคเอกชน
(โรงแรม/ห้าง)		 ภาคประชาสังคม
(ผู้นำชุมชน)
ระยะที่ 1: เตรียมการและวางแผน
1.1 จัดตั้งทีมงานและวางแผนภาพรวม A R C I I
1.2 ประชุมเปิดตัวและระดมภาคีเครือข่าย R C A C C
1.3 คัดเลือกพื้นที่นำร่องและวิเคราะห์ข้อมูล A R C I I
ระยะที่ 2: ดำเนินการและปฏิบัติการ
2.1 จัดอบรมให้ความรู้ (ผู้ประกอบการ) A R C R I
2.2 จัดอบรมให้ความรู้ (ครัวเรือน) I C A I R
2.3 วิเคราะห์องค์ประกอบขยะ ณ สถานที่กำจัด A R C - -
ระยะที่ 3: ติดตามผลและขยายผล
3.1 ประเมินผลและสรุปบทเรียนโครงการ A R C C C
3.2 จัดทำคู่มือและสื่อประชาสัมพันธ์ A R I I I

คำอธิบายสัญลักษณ์

R
Responsible (ผู้รับผิดชอบหลัก):
ผู้ลงมือทำกิจกรรมนั้นๆ ให้สำเร็จ
A
Accountable (ผู้มีอำนาจตัดสินใจ):
เจ้าของกิจกรรม มีอำนาจอนุมัติและรับผิดชอบผลลัพธ์ (มีได้คนเดียว)
C
Consulted (ผู้ให้คำปรึกษา):
ผู้ที่ต้องให้ข้อมูลหรือความเห็นก่อนลงมือทำ
I
Informed (ผู้ที่ต้องรับทราบ):
ผู้ที่ต้องได้รับแจ้งความคืบหน้าหรือผลลัพธ์
